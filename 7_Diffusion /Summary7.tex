\section*{What the Notebook Does}

This notebook completes a template for a Denoising Diffusion Probabilistic Model (DDPM) to generate handwritten MNIST digits using the \texttt{denoising\_diffusion\_pytorch} library. It implements the core training mechanics, including calculating the diffusion loss via forward propagation, performing backpropagation, and tracking the loss over time. It also generates and visualizes intermediate digit samples to monitor the model's progress.

\section*{Architecture Choices}

The main design choices revolve around the predefined U-Net configuration, the DDIM sampling, and the implementation of the training loop:

\begin{itemize}
    \item \textbf{U-Net and DDIM:} The core model uses a custom U-Net architecture with dimension multipliers \texttt{(1, 2, 5)} to capture complex spatial patterns in the 28x28 images. The evaluation process is accelerated by reducing the reverse sampling timesteps from 1000 to 250 using Denoising Diffusion Implicit Models (DDIM).
    \item \textbf{Training loop implementation:} The core training mechanics are added to an empty template loop. This involves passing the batch of images through the \texttt{diffusion} function to calculate the loss, followed by standard PyTorch backpropagation and AdamW optimizer steps.
\end{itemize}
